[ HTML 기초 ]

1. HTML은 Hyper Text Markup Language를 나타냅니다.

: 하이퍼텍스트(Hyper Text)란?
- 순서가 없는 정보를 담고 있는 텍스트
: 마크업(Markup) 이란?
- 정보를 구조화하여 순서대로 웹
페이지에 띄우는 것(표시하는 것)

-> Hyper Text Markup Language란?
순서가 없는 정보를 담고있는 텍스트의
정보를 구조화하여 순서대로 웹 페이지에 표시하기 위한 브라우저 언어

[ 구조화란? ]
- 구조 -> 체계 -> 순서/분류
-> 체계가 없는 문자 데이터를
체계있게 순서를 정해 주는것!

2. HTML은 XML에서 온 것이다.
- XML은 정보를 구조화하는 확장가능한 마크업 언어
- 표기법: 
	1) 최상단에 xml선언부
	2) 최상위 태그 하나만 생성
	3) 하위 구조화 태그 생성

(참고: 파일의 확장자란?)
- 파일포멧을 알려주는 표시자
※ 파일의 확장자를 반드시 보이게할것!

3. HTML의 태그 구조
- 태그: 구조화요소의 표시자
- 태그(Tag)는 요소(Element)라고도 함

1) 선언부
-> 느껴! 문서타입 html
<!doctype html>
-> 브라우저가 이것을 읽는 순간
html5문법으로 읽을 준비를 한다!

2) 전체부모(html) : 오직하나!
-> 이것을 다른말로 Root라고함!
<html></html>

3) 머리(head)
-> 웹페이지 타이틀, 인코딩설정 등
<head></head>
   3-1) meta태그: 문서의 기본정보표시
   - charset="utf-8"
   -> 한글, 일본어 등 2byte체계의
문자셋이 잘 표시되도록 인코딩설정함
   
   3-2) title태그: 브라우저 탭에 제목출력

4) 몸(body)
-> 브라우저에 출력되는 내용
<body></body>

4. 저장시 확장자
-> .html 또는 .htm 
- 이것이 브라우저가 읽을 수 있는 
웹문서 확장자!
- 이름이 같아도 확장자가 다르면
다른문서로 인식함
