[ 웹디기능사 실기 작업순서 ]

1. 결과파일 폴더에 하위폴더 3개를 생성함
  1) images
  2) css
  3) js

2. index.html파일을 만들고 기본 틀(레이아웃)을 코딩함

3. css 연결, js연결후 연결여부확인
 1) CSS 는 body배경색 넣기로 확인
 2) JS는 console.log에서 연결확인함

4. 레이아웃 html을 작성 후 CSS레이아웃 완성!

5. 상세코딩
  1) GNB 메뉴
  - 유형에 따라 코딩법을 적절히 적용한다
  -> 가로메뉴 : 
    하위 전체내려옴/하위 하나씩 내려옴/하위 전체 배경같이 내려옴
  -> 세로메뉴 :
    하위 아래로 밀고 내려옴 / 하위 오른쪽 사이드에서 따로 내려옴
  ※ GNB메뉴가 속한 상단영역의 z-index를 가장 높게 설정한다!
    (하위메뉴가 다른 영역 밑으로 내려가지 않도록 조치함)
    - z-index가 적용되려면 position이 static이 아니어야함!

  2) 이미지 작업 : 각 영역별 사이즈에 맞게 조건에 맞게 제작한다
  - 슬라이드와 같이 같은 사이즈로 여러개를 만들경우
    캔버스크기를 여러개만큼 크게하여 모두 제작후 슬라이스로 잘라
    한번에 저장하는게 편리하다!(슬라이스툴 등분할 기능 사용하기!)
  - 이미지작업은 최대 40분이내로 작업한다!

  3) 로고 영역 코딩 +  하단로고까지 한꺼번에 코딩
  4) 슬라이드 코딩:
    - 유형에 따라 html, css, js를 적절히 적용한다.
  5) 컨텐츠 영역 코딩
    - 공지사항/갤러리 탭메뉴 출제시 js까지 모두 코딩한다.+팝업창코딩
    - 배너, 바로가기는 간단히 이미지만 조절하여 크기를 맞춘다!
  6) footer 영역 코딩 : 카피라잇, SNS, 하단메뉴 등 위치를 잘 맞춘다

6. 전체 셋팅상 유의사항:
  1) 파티션 박스에 패딩과 보더박스 셋팅을 추가한다!
  2) h1,h2 태그 등 마진,패딩 0 설정필수!
  3) a태그로 메뉴, 슬라이드, 배너 등 반드시 감싸준다.(href="#"필수)
  4) 영역에 적절히 배경색을 디자인을 최소화한다!(보더는 지양)
  5) 자바스크립트/제이쿼리는 철저히 연습하여 에러가 절대없어야함
  -> JS작업이 없으면 실격! 에러나도 실격!
