- 다운로드 설치후
1. 확장프로그램 설치

 2) 검색으로 확장 플러그인 선택
 3) install(설치)버튼으로 선택

   : 모든 메뉴 등 한국어 지원

   : 이미지 미리보기 기능 지원
 3) Live Server
   : 브라우저 실시간 미리보기
 4) Beautify : 자동줄맞춤
  -> 단축키 지정하기

[ vscode 사용법 + 단축키 ]

1. 작업폴더 선택하기
   파일>폴더열기

2. 소스파일 열기
 - 임시열기: 한번클릭(탭 파일명이 이탤릭체)
 - 확정열기: 더블클릭(탭 파일명이 노멀체)

3. 테마(화면색, 글자색) 변경하기
 - F1을 눌러서 검색이 나오면 "theme"이라고 친다
 - "기본설정 색테마"를 선택한다.
 - 제시된 기본 옵션중 테마를 선택한다.

4. 코드 탐색기(왼쪽 네비게이션)
 - 왼쪽 메뉴 첫번째 버튼 토클클릭
 - 단축키: ctrl+B

5. 화면 확대/축소
 - 단축키: ctrl+플러스키 (확대)
	ctrl+마이너스키(축소)

6. 줄바꿈 토글(보기메뉴에 있음)
- 단축키: alt+Z(알지?)
- 라인번호 확인필수(한줄코드는 같은번호에 있음)

7. 코딩시 소스선택(html, css)
